# DNDT

## requirements

- python3
- numpy
- matplotlib
- sklearn
- torch
- pandas
- functools

## usage

```bash
python3 DNDT.py
python3 DT.py
```

## Warning

- **The hyperparameters are not tuned, so the results may be different from the paper.**
- **The seed is not fixed, so you may get different results each time you run the code.**
